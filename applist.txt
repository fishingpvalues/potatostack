## INTEGRATION HIGHLIGHTS

### Database Consolidation
- Postgres: Nextcloud, Authentik, Gitea, Woodpecker, Immich, Calibre, Linkding, n8n, Healthchecks, Stirling PDF, Atuin, Homarr, Paperless, Open WebUI, Miniflux, Grafana
- Redis: Shared cache, Nextcloud, Immich
- MongoDB: General document storage
- CouchDB: Obsidian LiveSync

### VPN Protection
All behind Gluetun VPN:
- *arr stack (Sonarr, Radarr, Lidarr, Readarr, Prowlarr, Bazarr)
- qBittorrent
- Aria2

### Monitoring Integration
Prometheus scrapes:
- Node Exporter (system metrics)
- cAdvisor (container metrics)
- Fritzbox Exporter (router metrics)
- Netdata (comprehensive metrics)
- All services expose metrics on /metrics endpoint

Loki collects logs from:
- All Docker containers via Promtail
- System logs (/var/log)

Grafana visualizes:
- Prometheus metrics (SOTA dashboards provisioned)
- Loki logs
- Pre-configured dashboards for all services

### Reverse Proxy
Traefik handles:
- Automatic SSL via Let's Encrypt
- Docker service discovery
- Load balancing
- Header middleware

Nginx Proxy Manager provides:
- GUI-based proxy configuration
- Alternative to Traefik
- SSL management

### Automation Workflows
n8n integrates with:
- All services via APIs
- Webhooks
- Database access
- File system access

Huginn provides:
- IFTTT-like automation
- Web scraping
- RSS aggregation

Healthchecks monitors:
- Cron jobs
- Backup tasks
- Service health

### SSO Ready
Authentik configured for:
- OIDC/SAML provider
- All services can integrate
- Centralized authentication
- 2FA enforcement
