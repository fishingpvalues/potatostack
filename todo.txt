Use oleduc/docker-obsidian-livesync-couchdb (automates admin user, databases, CORS, etc.):
YAMLservices:
  obsidian-livesync:
    image: oleduc/docker-obsidian-livesync-couchdb:latest
    container_name: obsidian-livesync
    environment:
      - COUCHDB_USER=obsidian
      - COUCHDB_PASSWORD=SuperSecretPass123!
      - COUCHDB_DATABASE=obsidian-vault
    volumes:
      - ./couchdb-data:/opt/couchdb/data
    ports:
      - "5984:5984"
    restart: unless-stopped
Client Setup (on every device)

Install the Self-hosted LiveSync community plugin in Obsidian
Go to plugin settings → Self-hosted LiveSync
Use the Setup URI generator (or enter manually):
CouchDB URL: http://your-server-ip:5984 (or HTTPS via reverse proxy)
Username + Password
Database name (usually obsidian-vault)

Enable "LiveSync" and "Periodic sync"

Strongly recommended: Put it behind a reverse proxy (Traefik/Nginx/Caddy) with HTTPS + authentication.
